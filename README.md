# 📊 Bellabeat Case Study – Data Analysis Project

## 🔍 Project Overview
The Bellabeat project analyzes smart device usage data to understand user activity patterns, sleep behavior, and recovery trends. The goal is to generate data-driven insights and provide actionable recommendations to improve user engagement and overall wellness for Bellabeat.

---

## 🎯 Business Task
Bellabeat, a wellness company focused on women's health, wants to understand how users interact with their smart devices.  
This analysis aims to identify behavioural patterns and translate them into strategies that enhance user experience and support business growth.

---

## 📁 Dataset
- **Source:** Kaggle – Fitbit Fitness Tracker Data  
- **Time Period:** March 2016 – May 2016  
- **Users:** 35 consenting users (note: user count varies across tables, indicating partial data coverage)

The dataset includes:
- Daily activity (steps, calories)
- Sleep data
- Heart rate data  

---

## 🛠️ Tools Used
- **Microsoft Excel** – data cleaning and preparation  
- **SQL (Google BigQuery)** – data querying and analysis
- **MySQL Workbench** – used for executing queries, particularly for large heart rate datasets   
- **Command-line tools** – handling and merging large CSV files (~3.6 million records)  
- **Tableau** – data visualization and dashboard creation

*SQL queries used for analysis are included in the repository*

## 📌 Process
1. **Ask** – Defined the business problem and objectives  
2. **Prepare** – Collected and assessed the dataset  
3. **Process** – Cleaned and transformed the data  
4. **Analyze** – Identified trends and patterns  
5. **Share** – Created interactive dashboards  
6. **Act** – Provided business recommendations  

---

## 📊 Key Insights
1. Users are moderately active(7,281 steps) but do not reach recommended daily activity levels  
2. Activity peaks during the evening (4–7 PM)  
3. Higher night heart rate is associated with reduced sleep duration (R² = 0.66)  
4. Weekend engagement drops, especially on Sundays  

---

## 💡 Business Recommendations
**Engagement Strategy**
- Bellabeat can enhance its IVY Wellness Coach App by introducing personalised step goals based on user behaviour
- Implement gamification features such as streaks and rewards
- Send targeted notifications during peak activity hours
- Improve weekend engagement through activity prompts

**Recovery and Wellness Strategy**
- Develop a recovery score combining sleep and heart rate data
- Introduce wind-down reminders to improve sleep quality
- Offer premium recovery insights to drive subscription value
  
---

## 📈 Expected Business Impact
- 8–12% increase in user engagement *(based on industry benchmarks for gamification features)* 
- 5–7% growth in premium subscriptions  
- Improved user retention through personalization
